# Add a Custom Service Key
To add a custom record to your FNS name, open the name’s profile editor by clicking `Edit profile`

<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-add-custom-record-1.png"/>

In the FNS Manager app, there are common account service records you can add to your FNS name’s profile. For instance, GitHub, Twitter, and Discord can easily be added by clicking on the `Accounts` tab.

<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-add-custom-record-2.png"/>

What if you want to create a custom record for an Account service not listed already?
For example, you want to save your YouTube channel name into your FNS records. To set a custom record, click the tab named `Other`.

<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-add-custom-record-3.png"/>

Since YouTube is not listed as an account service by default, we will need to save a custom service key and value pair. Service keys are in the reverse dot format.

Therefore, click `+ Add Record` and type in the record name field as: `com.youtube`. Next, click `Add`  

:::note
Not every custom key should use the reverse dot format. Keys can be named anything, but account service keys should follow a standard reverse dot convention.
:::

<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-add-custom-record-4.png"/>

After clicking `Add`, a new line to input the value for the record will show. In this field, type in your YouTube channel name or user.

<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-add-custom-record-5.png"/>

Finally, click `Save`. This will open a wallet request to save the data to the blockchain.
Now a record for YouTube.com will exist under your FNS name! There is no limit to the amount or types of records you can use.
