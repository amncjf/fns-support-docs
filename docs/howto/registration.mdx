# How to register a name

Go to the <Link to="https://app.fildomains.com"><b>FNS Manager App</b></Link> and connect your wallet.

## Search for the FNS name you want

<BrowserWindow url="https://app.fildomains.com" padding="125px" bg="#f0f3f7" img="/img/howto/v3-register-1_test.png"/>

Provided that the name you searched for is available, click on it to continue to the beginning of the registration.

<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f3f4f7" img="/img/howto/v3-register-2.png"/>

Select the amount of years you want to register the name for, and if you'd like to set it as your wallets `Primary name`
during the registration process and then click `Next` 

:::info
For more information about what the `Primary name` does see:
 - <Link to="/core/records/primary-name"><b>Records:</b> Primary Name</Link>
:::

## Registering your FNS name
Registering an FNS name is a three step process:
:::info Transaction details
```mermaid
%%{
  init: {
    'flowchart': {
      'curve': 'linear'
      } } }%%
      
graph LR;
A(Request to Register);
B(Wait 60 Seconds);
C(Register);

classDef active stroke-width:4px;
classDef inactive stroke-dasharray: 5 5;

A-->B-->C;
```

 1. `Request to Register`  
 *A 0ETH commit transaction to prevent front-running*
   
 1. `Wait 60 Seconds`  
 *A 60 second waiting time*
   
 1. `Register`  
 *The actual Registration transaction*
:::

### Step 1: Request to register
```mermaid
%%{
  init: {
    'flowchart': {
      'curve': 'linear'
      } } }%%
      
graph LR;
A(Request to Register);
B(Wait 60 Seconds);
C(Register);

classDef active stroke-width:4px;
classDef inactive stroke-dasharray: 5 5;

class A active;

A-->B-->C;
```
Click `Request to Register` to start **Step 1** of the registration process.
A 0 FIL transaction is performed where your name is hashed with a secret key so that no one else can view the name you're trying to register.

:::info
While this transaction does not have a transaction value, it still incurs a gas fee.
:::

:::caution
* This step stores a *secret key* in your browser's local storage. To avoid having to repeat this step, make sure to ***not*** clear your cache or switch browsers.

* You have ***7 days*** to complete Step 3 before the `Request to Register` transaction expires. *However,* the FNS name is ***not reserved for you*** during this time.
:::

Double-check that the registration length and fees are satisfactory, then press `Begin` to start the registration process.

<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f3f4f7" img="/img/howto/v3-register-cthulu-11.png"/>

Click `Open wallet` and approve the transaction in your wallet.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f3f4f7" img="/img/howto/v3-register-cthulu-12.png"/>

Wait while the transaction is processed by the Filecoin blockchain.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f3f4f7" img="/img/howto/v3-register-cthulu-13.png"/>

### Step 2: Wait 60 seconds
```mermaid
%%{
  init: {
    'flowchart': {
      'curve': 'linear'
      } } }%%
      
graph LR;
A(Request to Register);
B(Wait 60 Seconds);
C(Register);

classDef active stroke-width:4px;
classDef inactive stroke-dasharray: 5 5;

class B active;

A-->B-->C;
```
After the Step 1 transaction completes there's a 60-second waiting period to prevent front running.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f3f4f7" img="/img/howto/v3-register-cthulu-14.png"/>
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f3f4f7" img="/img/howto/v3-register-cthulu-15.png"/>

### Step 3: Register
```mermaid
%%{
  init: {
    'flowchart': {
      'curve': 'linear'
      } } }%%
      
graph LR;
A(Request to Register);
B(Wait 60 Seconds);
C(Register);

classDef active stroke-width:4px;
classDef inactive stroke-dasharray: 5 5;

class A,B inactive;
class C active;

A-->B-->C;
```
Now it's time to `Register` your FNS name. Click the register button and confirm the transaction in your wallet.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f3f4f7" img="/img/howto/v3-register-cthulu-16.png"/>
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f3f4f7" img="/img/howto/v3-register-cthulu-17.png"/>

Congratulations! If all transactions went through successfully you should now be the proud owner of your very own FNS name!
