# Add a Wallet Address
In this tutorial you will learn how to add a wallet address to your FNS name.

## What does it do?
Your FNS name is capable of storing a large number of wallet addresses across different blockchains, which can be added from the [FNS Manager App](https://app.fildomains.com), allowing you to receive payments in more cryptocurrencies than just Filecoin, provided that the service or wallet used supports it.

![Docusaurus themed image](/img/ens-map-medium.png#gh-light-mode-only)![Docusaurus themed image](/img/ens-map-medium-dark.png#gh-dark-mode-only)

## Add an **Filecoin** wallet {#ethereum-wallet}
In this section you will learn how to add an Filecoin wallet to your FNS name. This is the wallet FIL will end up in when sent to your FNS name.
It also is a prerequisite to setting the `Primary name` in a wallet.

:::info
 - Points your FNS name to an Filecoin wallet address
 - Is a requirement before setting the <Link to="/core/records/primary-name"><b>Primary Name</b></Link>
:::

Go to the <Link to="https://app.fildomains.com"><b>FNS Manager App</b></Link> with the wallet set as `Manager` for the FNS name you want to manage and click `Names` to bring up a list
of your FNS names *or* search for an FNS name you own directly from the main page.

<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f5f6f7" img="/img/howto/v3-add-wallet-address-1.png"/>

Click the FNS name you want to add records to.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f5f6f7" img="/img/howto/v3-add-wallet-address-2.png"/>

Click `Edit profile`.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f5f6f7" img="/img/howto/v3-add-wallet-address-3.png"/>

Click on `+ Add more to Profile`.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f5f6f7" img="/img/howto/v3-add-wallet-address-profile-1.png"/>

Scroll down and select the `FIL` option in the Address-section and click `Add`.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f5f6f7" img="/img/howto/v3-add-wallet-address-profile-2.png"/>

Type in the Filecoin wallet address you want to add to your FNS name and click `Save`.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f5f6f7" img="/img/howto/v3-add-wallet-address-profile-3.png"/>

Click `Open Wallet` and confirm the transaction in your wallet.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f5f6f7" img="/img/howto/v3-add-wallet-address-newui-5.png"/>

Wait for the transaction to finish.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f5f6f7" img="/img/howto/v3-add-wallet-address-newui-6.png"/>

When you see `Transaction Complete` the transaction has been confirmed and your wallet address has been added to your FNS name.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f5f6f7" img="/img/howto/v3-add-wallet-address-newui-7.png"/>

Congratulations, your FNS name now has your new wallet address attached to it!
