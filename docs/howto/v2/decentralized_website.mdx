# Create a decentralized website

---

One of the most interesting and powerful features of FNS names is the ability to point them to webpages hosted on the decentralized web, creating a website where both the hosting and the domain name is *entirely* decentralized.

In this tutorial you will:
 1. [**Build a website**](#build-a-website)
 1. [**Set up a GitHub repository**](#set-up-a-github-repository)
 1. [**Upload it to a decentralized storage using IPFS**](#upload-to-ipfs)
 1. [**Point your FNS name to it**](#configure-your-ens-name)
 
_ENS supports several decentralized protocols apart from IPFS, but this tutorial only covers IPFS (for now)._

Anyhow, let's get started!

## Build a website
In this section we'll create a basic website to deploy to IPFS using <Link to="http://heavens.tools"><b>Heaven's Tools</b></Link> excellent Web3 website builder.
:::tip
If you already have a website you'd like to upload, proceed to [**Set up a GitHub repository**](#set-up-a-github-repository)
:::

### Create the website
Before we can upload a site to IPFS we have to first create it! There are many ways to do that, but in this guide we'll try to keep it simple by using Heaven's Tools which is a WYSIWYG, or "What You See Is What You Get", type editor where you don't have to know how to code.

All you need to do is go to <Link to="http://heavens.tools"><b>Heaven's Tools</b></Link> website and drag blocks in the order you want them and Heaven's Tools will take care of the rest and build the site for you!

## Upload to IPFS
IPFS, or the _InterPlanetary FileSystem_, is a decentralized storing protocol we'll use to store our website files in, and serve them from.
For this tutorial we'll use a so called _"pinning service"_ to store our files on IPFS but it's also possible to pin files yourself using a local IPFS node.

The benefit of using a pinning service is that you'll not need to run your own IPFS node to make sure that your website is accessible.

### Create an account with Fleek
Go to <Link to="https://fleek.co"><b>Fleek's website</b></Link> and `Sign Up` for an account.

### Upload your site to Fleek
After logging into <Link to="https://fleek.co"><b>Fleek</b></Link>, at the dashboard select `Add new site`.
<BrowserWindow url="fleek.co">
  <Image img="/img/howto/create_website_fleek_1.png" />
</BrowserWindow>

Choose `Connect with GitHub` and connect your GitHub account. 
<BrowserWindow url="fleek.co">
  <Image img="/img/howto/create_website_fleek_2.png" />
</BrowserWindow>

Click `Install & Authorize`
<BrowserWindow url="fleek.co">
  <Image img="/img/howto/create_website_fleek_3.png" />
</BrowserWindow>

When asked which repository to create the website from, select the repository you created earlier in [](#step-1-create-a-repository)
<BrowserWindow url="fleek.co">
  <Image img="/img/howto/create_website_fleek_4.png" />
</BrowserWindow>

For the hosting service, make sure IPFS is selected and press `Continue`.
<BrowserWindow url="fleek.co">
  <Image img="/img/howto/create_website_fleek_5.png" />
</BrowserWindow>

On the next screen no changes are necessary, so just scroll down and click `Deploy site`.
<BrowserWindow url="fleek.co">
  <Image img="/img/howto/create_website_fleek_6.png" />
</BrowserWindow>

Fleek will now deploy the website to IPFS, which might take a little while.
<BrowserWindow url="fleek.co">
  <Image img="/img/howto/create_website_fleek_7.png" />
</BrowserWindow>

When the site has been deployed, it will look like this:
<BrowserWindow url="fleek.co">
  <Image img="/img/howto/create_website_fleek_8.png" />
</BrowserWindow>

Select the `IPFS CID` *(marked in blue below)* and press `CTRL + C` to copy it into the clipboard or write it down.
<BrowserWindow url="fleek.co">
  <Image img="/img/howto/create_website_fleek_9.png" />
</BrowserWindow>

## Configure your FNS name

### Go to the FNS Manager App
Go to the <Link to="https://app.fildomains.com"><b>FNS Manager App</b></Link> and connect your wallet.
<BrowserWindow url="app.fildomains.com">
  <Image img="/img/howto/create_website_ens_1.png"/>
</BrowserWindow>

Click on `My Account` to go to a list of your FNS names.
<BrowserWindow url="app.fildomains.com">
  <Image img="/img/howto/create_website_ens_2.png"/>
</BrowserWindow>

Go to your FNS name and click `Add/Edit Record`, scroll down and enter the `IPFS CID` you saved earlier into the `Content` field, then scroll down and click `Save`
<BrowserWindow url="app.fildomains.com">
  <Image img="/img/howto/create_website_ens_3.png"/>
</BrowserWindow>

<BrowserWindow url="app.fildomains.com">
  <Image img="/img/howto/create_website_ens_4.png"/>
</BrowserWindow>
