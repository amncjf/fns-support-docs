# How to register a name

## Go to the FNS Manager App
Go to the <Link to="https://app.fildomains.com"><b>FNS Manager App</b></Link> and connect your wallet.

## Search for the FNS name you want

</BrowserWindow>

Provided that the name you searched for is available, click on it to continue to the registration.
<BrowserWindow url="app.fildomains.com">
  <Image img="/img/register_name_img4.webp"/>
</BrowserWindow>

## Registering your FNS name
Registering an FNS name is a three step process:
```mermaid
%%{
  init: {
    'flowchart': {
      'curve': 'linear'
      } } }%%
      
graph LR;
A(Request to Register);
B(Wait 60 Seconds);
C(Register);

classDef active stroke-width:4px;
classDef inactive stroke-dasharray: 5 5;

A-->B-->C;
```

:::info Transaction details
| Transaction           | Details                                              |
| --------------------- | ---------------------------------------------------- |
| `Request to Register` | *A 0 FIL commit transaction to prevent frontrunning* |
| `Wait 60 seconds`     | *A 60 second waiting time*                           |
| `Register`            | *The actual registration transaction*                |
:::

### Step 1: Request to register
```mermaid
%%{
  init: {
    'flowchart': {
      'curve': 'linear'
      } } }%%
      
graph LR;
A(Request to Register);
B(Wait 60 Seconds);
C(Register);

classDef active stroke-width:4px;
classDef inactive stroke-dasharray: 5 5;

class A active;

A-->B-->C;
```
Click `Request to Register` to start **Step 1** of the registration process.
A 0 FIL transaction is performed where your name is hashed with a secret key so that no one else can view the name you're trying to register.

:::info
While this transaction does not have a transaction value, it still incurs a gas fee.
:::

:::caution
* This step stores a *secret key* in your browser's local storage. To avoid having to repeat this step, make sure to ***not*** clear your cache or switch browsers.

* You have ***7 days*** to complete Step 3 before the `Request to Register` transaction expires. *However,* the FNS name is ***not reserved for you*** during this time.
:::

<BrowserWindow url="app.fildomains.com">
  <Image img="/img/register_name_img5.webp"/>
</BrowserWindow>

Verify that the cost of the transaction is what you expect it to be and confirm the transaction in your wallet.

### Step 2: Wait 60 seconds
```mermaid
%%{
  init: {
    'flowchart': {
      'curve': 'linear'
      } } }%%
      
graph LR;
A(Request to Register);
B(Wait 60 Seconds);
C(Register);

classDef active stroke-width:4px;
classDef inactive stroke-dasharray: 5 5;

class A inactive;
class B active;

A-->B-->C;
```
After the Step 1 transaction completes there's a 60-second waiting period to prevent front running.
<BrowserWindow url="app.fildomains.com">
  <Image img="/img/register_name_img6.webp"/>
</BrowserWindow>

### Step 3: Register
```mermaid
%%{
  init: {
    'flowchart': {
      'curve': 'linear'
      } } }%%
      
graph LR;
A(Request to Register);
B(Wait 60 Seconds);
C(Register);

classDef active stroke-width:4px;
classDef inactive stroke-dasharray: 5 5;

class A,B inactive;
class C active;

A-->B-->C;
```
Now it's time to `Register` your FNS name. Click the register button and confirm the transaction in your wallet.
<BrowserWindow url="app.fildomains.com">
  <Image img="/img/register_name_img7.webp"/>
</BrowserWindow>

Congratulations! If all transactions went through successfully you should now be the proud owner of your very own FNS name!
