# Transfer an FNS name

In this tutorial you will learn how to transfer an FNS name to another wallet.

When transferring an FNS name, only the `Owner` record is automatically updated to reflect the new owner wallet.
Depending on your specific use-case it's therefore a good idea to take a moment and think about what additional records you want to transfer.

There are three records we need to pay attention to when transferring a name:

:::info
 1. **Owner:** *The owner of the FNS name, the wallet the FNS name NFT resides in.*
 1. **Manager:** *The manager of records, the wallet allowed to make changes to its records (but not transfer the name)*
 1. **FIL Address:** *The wallet address the FNS name points to, where funds sent to it will arrive.*
:::

## Completely transfer a name
If you want to completely transfer a name to a wallet you'll need to set three records to the new wallet in a specific order because only the wallet set as `Manager` is allowed to change the `FIL Address`.
:::info For example
 - If you were to change `Manager` first, you wouldn't be able to set the `FIL Address` without switching to the new wallet.
:::

In order to set everything to the new wallet easily from the wallet that currently owns the FNS name, you'd follow this specific order:

:::info Order
 1. **FIL Address:** *The wallet address the FNS name points to, where funds sent to it will arrive.*
 1. **Manager:** *The manager of records, the wallet allowed to make changes to its records (but not transfer the name)*
 1. **Owner:** *The owner of the FNS name, the wallet the FNS name NFT resides in.*
:::

## Gift a Name with ENSFairy
If you would like to register a name on behalf of another wallet, <Link to="https://ensfairy.xyz"><b>ENSFairy</b></Link> is a great way to do that. This is because you can register a name with one wallet, but set up the ownership and address records to point to a different wallet.
