# How to set an Avatar

In this tutorial you will learn how to set an avatar on your FNS name.

## What does it do?

The avatar record allows you to select a custom background image for your FNS name `metadata` to display in services that support it.

## Set the Avatar
Go to the <Link to="https://app.fildomains.com"><b>FNS Manager App</b></Link> and make sure to connect with with the wallet set as `Manager` for the FNS name you want to manage and click `Names` to bring up a list
of your FNS names *or* search for an FNS name you own directly from the main page.

<BrowserWindow url="https://app.fildomains.com" padding="0px" bg="#f2f4f7" img="/img/howto/v3-set-avatar-1.png"/>

Click the FNS name you want to set the Avatar on.
<BrowserWindow url="https://app.fildomains.com" padding="0px" bg="#f2f4f7" img="/img/howto/v3-set-avatar-2.png"/>

Then click `Edit profile` to go to your profile settings.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-set-avatar-3.png"/>

Click the *"Camera"*-button to bring up a sub-menu of options to set your Avatar.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-set-avatar-4.png"/>

### Select how to set your avatar

:::info
In the sub-menu you have two options to set the NFT:
 1. Upload an image from your computer
 1. Select an NFT from your wallet
:::

<Tabs>
<TabItem value="Upload an image from your computer">
To upload an image file from your computer, click `Upload Image` and select an image file from your computer to upload.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-set-avatar-5.png"/>

Use the slider to crop the image to your liking and click `Continue`
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-set-avatar-6-upload-image-1.png"/>

Then click `Sign and Upload` to start the upload process.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-set-avatar-6-upload-image-2.png"/>

</TabItem>
</Tabs>

Click `Save` to save the changes
:::info
Note that this only saves them locally, and the next steps are required to publish it to the blockchain
:::

<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-set-avatar-7.png"/>

Confirm that the changes are correct, and then click `Open Wallet` and sign the transaction with your wallet
to modify the FNS name on-chain.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-set-avatar-8.png"/>

Once you've signed the transaction in your wallet, wait for the transaction to complete.
:::info
The speed of your transaction depends on Filecoin network congestion and the values set for gas on the transaction.
The higher you set the gas, the faster it will be performed.
If you set the gas too low, the transaction might stall and never complete.
:::
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-set-avatar-9.png"/>

When the transaction has completed your avatar is set. Click `Done` to go back to the FNS name page.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-set-avatar-10.png"/>

Congratulations! Your avatar is now set and should be visible from the FNS name page.
<BrowserWindow url="https://app.fildomains.com" padding="25px" bg="#f2f4f7" img="/img/howto/v3-set-avatar-11.png"/>

## Troubleshooting
If you're encountering problems you might find your issue addressed below!

### The image on the avatar doesn't cover the entire background
In order for the image set as the avatar to cover the entire background it needs to be square.
If the image appears on your avatar with a border on sides or on the top, that's because you've set an image that isn't square (of equal width and height).
