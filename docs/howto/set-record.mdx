Go to the <Link to="https://app.fildomains.com"><b>FNS Manager App</b></Link> with the wallet set as `Manager` for the FNS name you want to manage and click `Names` to bring up a list
of your FNS names *or* search for an FNS name you own directly from the main page.

<BrowserWindow url="https://app.fildomains.com" bg="#f3f4f7" img="/img/howto/v3-set-record-1.png"/>

Click the FNS name you want to add records to.

<BrowserWindow url="https://app.fildomains.com" bg="#f3f4f7" padding="25px" img="/img/howto/v3-set-record-2.png"/>

Go to the `Records` tab
<BrowserWindow url="https://app.fildomains.com" bg="#f3f4f7" padding="25px" img="/img/howto/v3-set-record-3.png"/>

Click `Edit Records`
<BrowserWindow url="https://app.fildomains.com" bg="#f3f4f7" padding="25px" img="/img/howto/v3-set-record-4.png"/>

Click `Add Record`
<BrowserWindow url="https://app.fildomains.com" bg="#f3f4f7" padding="25px" img="/img/howto/v3-set-record-5.png"/>

In the new field that appears, type in the name of your Custom Record. Records traditionally use a reverse notation, e.g. `com.youtube` for youtube.com
and then click `Add`

<BrowserWindow url="https://app.fildomains.com" bg="#f3f4f7" padding="25px" img="/img/howto/v3-set-record-6-text-2.png"/>

Type in a value for your new record, and click `Save`
<BrowserWindow url="https://app.fildomains.com" bg="#f3f4f7" padding="25px" img="/img/howto/v3-set-record-6-text-3.png"/>

Then click `Open Wallet` and approve the transaction in your wallet.
<BrowserWindow url="https://app.fildomains.com" bg="#f3f4f7" padding="25px" img="/img/howto/v3-set-record-6-text-4.png"/>

Wait for the transaction to complete.
<BrowserWindow url="https://app.fildomains.com" bg="#f3f4f7" padding="25px" img="/img/howto/v3-set-record-6-text-5.png"/>

Once the transaction has completed your record is stored on-chain and should be immediately accessible.
<BrowserWindow url="https://app.fildomains.com" bg="#f3f4f7" padding="25px" img="/img/howto/v3-set-record-6-text-6.png"/>
